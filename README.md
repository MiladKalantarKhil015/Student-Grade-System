# Student-Grade-System

Description: Console-based School Management System built using Java and Object-Oriented Programming principles.

## Features
- Add students
- Remove students
- Search students by ID
- Create courses
- Display students and courses
- Grade validation
- Letter grade system

## OOP Concepts Used
- Encapsulation
- Composition
- Classes and Objects
- Method separation

## Technologies
- Java
- OOP
- Arrays
- Console Application
